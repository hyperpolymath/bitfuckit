== What is bitfuckit?

`bitfuckit` is a command-line interface for Bitbucket Cloud, written in Ada/SPARK for reliability and formal verification.

=== Why Does This Exist?

Every major git forge has an official CLI:

* **GitHub** ‚Üí `gh`
* **GitLab** ‚Üí `glab`
* **Codeberg** ‚Üí `tea`
* **SourceHut** ‚Üí `hut`
* **Gitea** ‚Üí `tea`

**Bitbucket has nothing.** Atlassian, a company valued at over $50 billion, has not provided CLI tooling for their flagship version control product.

This tool fills that gap.

=== Key Features

* **Authentication** ‚Äî App password-based authentication
* **Repository CRUD** ‚Äî Create, list, delete, check existence
* **Pull Requests** ‚Äî List PRs with state filtering
* **Mirroring** ‚Äî One-command GitHub-to-Bitbucket mirroring
* **Interactive TUI** ‚Äî Vim-style navigation with color interface
* **GraphQL API** ‚Äî Programmatic access for automation
* **SPARK Verification** ‚Äî Formally verified TUI specification

== Quick Start

[source,bash]
----
# 1. Authenticate with Bitbucket
bitfuckit auth login

# 2. List your repositories
bitfuckit repo list

# 3. Create a new repository
bitfuckit repo create my-new-repo --private

# 4. Mirror from GitHub
cd my-github-repo
bitfuckit mirror my-repo

# 5. Launch the TUI
bitfuckit tui
----
