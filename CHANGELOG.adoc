== [0.1.0] - 2025-12-25

=== Added
* Initial release
* `auth login` - Authenticate with Bitbucket app password
* `auth status` - Show authentication status
* `repo create` - Create repositories (public/private, with description)
* `repo list` - List all repositories in workspace
* `repo delete` - Delete repositories (with confirmation)
* `repo exists` - Check if repository exists
* `mirror` - Mirror from GitHub to Bitbucket
* Interactive TUI mode with vim-style navigation
* SPARK-verified TUI specification
* Man page (`doc/bitfuckit.1`)
* Shell completions (bash, zsh, fish)
* Packaging files (RPM spec, Arch PKGBUILD, Nix flake)
* Justfile for common tasks
* SCM checkpoint files (META, ECOSYSTEM, STATE, AGENTIC, NEUROSYM, PLAYBOOK)
